# 썸네일 생성기

이미지에 제목을 추가하여 썸네일을 생성하는 간단한 프로그램입니다.

## 설치 방법

1. Python 3.7 이상이 설치되어 있어야 합니다.

2. 필요한 라이브러리 설치:
```bash
pip install -r requirements.txt
```

## 사용 방법

1. 프로그램 실행:
```bash
python thumbnail_generator.py
```

2. "이미지 선택" 버튼을 클릭하여 원본 이미지를 선택합니다.

3. 제목 입력란에 썸네일에 표시할 제목을 입력합니다.

4. "썸네일 생성" 버튼을 클릭합니다.

5. 저장할 위치와 파일명을 선택하면 썸네일이 생성됩니다.

## 기능

- 다양한 이미지 포맷 지원 (JPG, PNG, BMP, GIF)
- 이미지 크기에 맞춰 자동으로 폰트 크기 조정
- 텍스트 가독성을 위한 배경 및 외곽선 효과
- 하단 중앙에 제목 배치
- 고품질 이미지 저장
